# /plan-validate

Você está na etapa 5 de 5 do refinamento de plano.

> Os arquivos `PLAN.md` e `HANDOFF.md` ficam na **raiz do projeto** (mesma pasta onde este projeto é aberto). Sempre leia e salve a partir de lá.

## Sua tarefa

1. Leia o `PLAN.md` atual (na raiz do projeto)
2. Releia cada etapa simulando ser um agente com pouca capacidade de inferência:
   - Algum passo exige contexto que não está escrito? Reescreva.
   - Algum passo tem mais de uma interpretação? Reescreva.
   - Algum passo depende de outro sem deixar isso explícito? Corrija.
3. Faça os ajustes finais diretamente no `PLAN.md` (na raiz do projeto)
4. Salve o `PLAN.md` com o campo de status atualizado:

```
status: pronto
```

## Ao finalizar

Atualize o `HANDOFF.md` (na raiz do projeto) com o seguinte conteúdo:

```
# Handoff

etapa_concluida: 5-validate
proximo_comando: nenhum
status: pronto

## Resumo da sessão
[descreva em 2-3 linhas o que foi ajustado]

## Plano aprovado
O PLAN.md está pronto para execução por qualquer agente.
```

Salve ambos os arquivos e informe ao usuário que o plano está finalizado e pronto para execução.
